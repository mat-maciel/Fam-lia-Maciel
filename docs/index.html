Conecte ao Supabase

O app está no ar, mas ainda não tem um banco de dados para sincronizar entre todos. Faça uma vez:

  1. Crie um projeto grátis em supabase.com.
  2. No SQL Editor, rode o script docs/schema.sql deste repositório.
  3. Em Project Settings → API, copie a Project URL e a chave anon public.
  4. Cole as duas em docs/config.js e faça commit.
